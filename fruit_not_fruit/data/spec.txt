[Fruit not fruit dataset]
Input Features 32 * 32 * 3 RGB images of Fruits/ Not Fruits
Output Labels  1 - fruit, 0 - not fruit

Generation
1. Download and unzip Fruits-360 dataset into data folder:
    https://www.kaggle.com/moltean/fruits/downloads/fruits.zip/27
2. Download and unzip Cifar-100 dataset into data folder:
    http://www.cs.utoronto.ca/%7Ekriz/cifar-100-python.tar.gz
3. cd data && python3 src/dataset.py to generate dataset
4. Generated dataset will be named as dataset.pickle
